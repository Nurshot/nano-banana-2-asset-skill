# Privacy notice

Nano Banana 2 Assets runs locally and collects no telemetry.

For an image-generation operation, the skill sends Google Antigravity only:

- the image prompt designed from the visible project context; and
- up to three reference images explicitly selected for that task.

The prompt may describe the target layout, brand colors, and nearby copy, but the runner does not transmit project source code, conversation databases, OAuth tokens, keychain contents, or unrelated files. Authentication is handled by the user's existing local Agy OAuth session. Google's terms and privacy practices apply to data processed by Antigravity.
